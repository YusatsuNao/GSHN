How to use this Artifact Generator ::

INFO This artifact value is based on Artifact Optimizer, if you already have a good set and you want to try it in other "one" just type based on artifact you have
INFO You only can press Sub-stats 8 Times, because the game have a character limit, if you want add more than 8 you need to use command
INFO If you want to add same substat but multiple times just add "," after the ID, EXAMPLE : 501221,2 for Crit Damage 5.2% x 2 = 10.4 %
